Frequently Asked Questions (FAQ)

General

What is Cryon?
Cryon is a quantum-resistant cryptocurrency designed to provide security and simplicity in the post-quantum era.
How can I get started with Cryon?
Start by creating a wallet using our GUI or CLI tools. Then mint test tokens, try transfers, and participate in staking and bounty programs.
Is Cryon safe to use?
Cryon employs advanced post-quantum cryptographic algorithms. However, the project is currently in testnet phase and should be used cautiously.

Staking & Rewards

What is staking in Cryon?
Staking means locking your CRY tokens to support network security and earn rewards based on the amount and duration of staking.
How do I stake and unstake tokens?
You can stake and unstake tokens using the Cryon wallet GUI or via REST API endpoints. Rewards accumulate over staking time.
Are there any lock-up periods?
Currently, there is a minimum staking period of 7 days to receive rewards, but unstaking is possible anytime after that.
How are staking rewards calculated?
Rewards are approximately 1% of the staked amount per month, prorated by time staked.

Testnet & Bounty Program

What is the Testnet Bounty program?
A program to reward active testnet users by granting points for wallet creation, minting, transfers, and staking activity.
How do I earn bounty points?
Points are awarded for various actions on the testnet, such as creating wallets, minting tokens, transferring tokens, and staking.
What are bounty points used for?
Points may be used to qualify for airdrops or other early contributor rewards once mainnet launches.

Wallet & Usage

How do I create a Cryon wallet?
You can create a wallet using our desktop GUI or CLI tools. The wallet securely generates and stores your keys using post-quantum cryptography.
Can I use Cryon tokens on exchanges?
Currently Cryon is in testnet phase and not listed on exchanges. Plans for liquidity and exchanges are part of the roadmap.
How can I access Cryon APIs?
The public REST API offers endpoints for balance, transaction history, minting, transfers, and staking. Authentication via JWT is required for sensitive operations.

Technology & Security

What makes Cryon quantum-resistant?
Cryon uses hash-based cryptography and Dilithium post-quantum digital signatures, avoiding vulnerable traditional cryptography.
Are private keys exposed on-chain?
No. Addresses are derived from the SHA-256 hash of the private key, which is never revealed or reconstructed on-chain.
Is Cryon audited?
The project is currently in development and testing. Security audits are planned prior to mainnet launch.

Roadmap & Future

What are the main milestones on Cryon's roadmap?
Key milestones include mainnet launch, smart contract simulation, third-party audits, token economics finalization, and decentralization.
How can I contribute to Cryon?
You can contribute by testing on the testnet, reporting issues, joining bounty programs, or contributing code on GitHub.
